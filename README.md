## 什么样的功能和项目适合做自动化测试
- 需求稳定。不会频繁变更的。
- 周期比较长。（优惠活动，中秋，端午。）
- 复用性高， 可以跨系统，跨平台使用的。

## pytest vs unittest
- pytest 功能很强，比 unittest 强。
- unittest 标准库，python 内置， pip install pytest


## 面试题：为什么用 pytest, 
pytest 的特性，有哪些好处？

- 可以自动收集用例。
- assert
- pytest 支持上千种插件，自己的插件系统
- pytest 用例筛选功能
- pytest 有失败重运行功能
- pytest 有非常灵活的测试夹具，setUp, tearDown, fixture
- pytest 和 unittest 是兼容的。
